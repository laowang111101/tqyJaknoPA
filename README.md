# 基于SpringBoot的网上招聘管理系统 java810

## 项目概述

本系统是一个基于SpringBoot框架开发的网上招聘管理系统，包括后台管理端和前台用户端。后台提供招聘信息管理、用户管理等功能；前台则为用户提供岗位浏览、投递简历等操作。

## 技术栈

- **后端**：Java
- **前端**：JavaScript + Vue
- **数据库**：MySQL 5.7
- **开发平台**：IDEA + VSCode
- **运行环境**：Windows 10/11

## 功能模块

### 后台管理

- **岗位管理**：录入、修改、查询岗位基本信息，如名称、地区、备注等。
- **类型管理**：管理岗位类型信息，包括类型名称等。
- **标签管理**：录入、修改、查询标签信息。
- **评论管理**：管理网站评论信息。
- **公司管理**：浏览和管理公司信息。
- **简历管理**：浏览和管理简历信息。
- **用户管理**：管理用户信息，包括新增、编辑和删除。
- **统计分析**：根据岗位活动数据和会员参与度进行统计和分析。
- **消息管理**：发布系统消息。
- **系统信息**：查看系统基本信息，如服务器信息、内存信息、CPU信息等。

### 前台用户

- **注册登录**：用户通过注册和登录使用系统。
- **门户浏览**：用户可浏览岗位列表，包括最新、最热、推荐岗位。
- **智能推荐**：详情页右侧提供热门岗位推荐。
- **用户中心**：用户资料修改、邮箱推送、消息查看。
- **我的投递**：查看已投递岗位信息。
- **模糊搜索**：支持岗位信息的模糊搜索。
- **岗位评论**：用户可在岗位详情页下评论。

## 系统角色

- 系统管理员：负责后台管理操作。
- 普通用户：进行岗位浏览、投递简历等操作。
